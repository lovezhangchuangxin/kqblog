# 欢迎来到我的博客

这是我使用 **Next.js 16** 和 **Tailwind CSS v4** 构建的个人博客。

## 技术栈

这个博客使用了以下技术：

- **Next.js 16** - React 框架，支持 App Router 和 Server Components
- **Tailwind CSS v4** - 实用优先的 CSS 框架
- **MDX** - 支持 JSX 语法的 Markdown
- **Shiki** - 高性能代码高亮

## 代码示例

```typescript
// 一个简单的 React 组件
function Greeting({ name }: { name: string }) {
  return <h1>Hello, {name}!</h1>;
}

export default Greeting;
```

## 为什么选择这个技术栈？

1. **性能优先** - Next.js 的静态生成确保极快的加载速度
2. **开发体验** - MDX 让写作更加灵活
3. **SEO 友好** - 完整的元数据支持
4. **主题支持** - 多种颜色主题随心切换
